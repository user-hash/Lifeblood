Adapters (left)Roslyn is the C# reference. TypeScript is the second proof. JSON graph is the universal bridge. Community builds Go, Python, Rust, Java via the JSON protocol.
Core (center)Pure domain + application orchestration. GraphDocument carries version, language, and adapter capabilities through the entire pipeline. All invariants enforced by tests.
Connectors (right)Context packs, instruction files, MCP server with 6 interactive tools, CLI with CI exit codes. They depend inward on Application ports, never on adapters.
